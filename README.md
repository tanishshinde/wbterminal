# World Bank Data Terminal

A Bloomberg-style dashboard for exploring World Bank development indicators. 

## Features

- Live data from the World Bank API (api.worldbank.org/v2)
- 60+ development indicators across 8 themes
- Multi-country comparison (up to 5 countries)
- Time series charts with interactive tooltips
- Resizable panel layout
- Global search with keyboard shortcut (⌘K)

## Development

```sh
npm install
npm run dev
```
